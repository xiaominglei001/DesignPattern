单例模式是一种常见的设计模式，在《Java与模式》一书中，阎宏博士对单例模式做了全面的总结。
单例模式分三种：懒汉式单例、饿汉式单例、登记式单例三种。单例模式有一下特点：
1、单例类只能有一个实例。
2、单例类必须自己自己创建自己的唯一实例。
3、单例类必须给所有其他对象提供这一实例。

单例模式的种类:
一、懒汉式单例(lazy singleton)
二、饿汉式单例(eager singleton)
三、登记式单例(register singleton)
登记式单例类是为了克服饿汉式单例类及懒汉式单例类均不可继承的缺点而设计的。
